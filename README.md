# Poker Brothers CMS

### Stack / Support
* KeystoneJS
* Mongo DB
* [Mlab](https://mlab.com)
* [Heroku](https://www.heroku.com/)

### Commands
* ``yarn dev`` - for development, with nodemon
* ``yarn start``

### Config

Create a ``dotenv`` file:

* ``DB_URI`` - MongoDB database server

### API
* ``/api/players`` - All players
